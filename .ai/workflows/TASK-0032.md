committed but have never been smoke-tested end-to-end — not on WSL, not on a VPS. Before handing off

   reachable over HTTPS at a real domain.

## Part A — Local WSL Smoke Test

### A1. Clean Slate

Tear down any running stack and purge volumes so migration ordering and seed data are verified from scratch.

```bash
cd ~/workspace/voltana_V2
docker compose down -v
docker system prune -f
```

If the bind-mount data dir exists from a previous test:
```bash
sudo rm -rf /var/lib/voltana/
```

### A2. Fresh Local Deploy

```bash
cd ~/workspace/voltana_V2
docker compose up -d --build
```

Wait for `migrate` to exit 0, then confirm all containers healthy:
```bash
docker compose ps
```

Expected: `postgres` healthy · `redis` healthy · `migrate` exited(0) · `api` healthy · `nginx` running.

### A3. TASK-0029 Feature Tests

All via `curl http://localhost` (or the web UI if a browser is available).

| # | Test | Expected |

## Part B — VPS Production Deployment

### B1. Provision VPS

| Spec | Value |
|---|---|
| Provider | Hetzner CX22 / DigitalOcean Basic / Linode Shared |
| OS | Ubuntu 24.04 LTS |
| vCPU | 2 |
| RAM | 4 GB |
| Disk | 40–80 GB SSD |
| Region | EU (Frankfurt / Helsinki) — lowest latency to Iran |
| IPv4 | Static (required) |

### B2. Initial SSH + Repo Clone

```bash
ssh root@<VPS_IP>
apt-get update -q && apt-get install -y git
mkdir -p /opt && cd /opt
git clone https://github.com/Ali-roohy/voltana-v2.git voltana
cd voltana
```

### B3. Bootstrap

```bash
bash scripts/bootstrap-vps-prod.sh
```

Installs: Docker Engine + Compose v2, Node.js 20 LTS, certbot, awscli, UFW (22/80/443), creates
`voltana` system user, `/var/lib/voltana/{postgres,backups}` data dirs.

Re-run after the repo clone to also install the systemd backup timer.

### B4. DNS

Point `A` record for `your_domain.com` to the VPS static IP. Wait for propagation:

```bash
# From dev machine or VPS:
dig +short your_domain.com
# Should return the VPS IP
```

### B5. Environment

```bash
cp /opt/voltana/.env.production.example /opt/voltana/.env
nano /opt/voltana/.env
```

Fill every `REPLACE_WITH_` field:

| Variable | How to generate |
|---|---|
| `POSTGRES_PASSWORD` | `openssl rand -hex 32` |
| `JWT_SECRET` | `openssl rand -hex 32` |
| `BALE_BOT_TOKEN` | BotFather on Bale |
| `BALE_BOT_USERNAME` | Your bot's username (no @) |
| `TELEGRAM_BOT_TOKEN` | @BotFather on Telegram |
| `TELEGRAM_BOT_USERNAME` | Your bot's username (no @) |
| `SMTP_PASSWORD` | Resend API key (`re_…`) |
| `SMTP_FROM` | `noreply@your_domain.com` |
| `DOMAIN` | `your_domain.com` |
| `APP_URL` | `https://your_domain.com` |

### B6. TLS Certificate

```bash
# Stop anything that might hold port 80
certbot certonly --standalone -d your_domain.com
```

Certs land at `/etc/letsencrypt/live/your_domain.com/`.

Add auto-renewal cron:

```bash
crontab -e
# Add:
0 3 * * * certbot renew --quiet --deploy-hook "docker compose -f /opt/voltana/docker-compose.yml -f /opt/voltana/docker-compose.prod.yml exec nginx nginx -s reload"
```

### B7. Systemd Service

```bash
cp /opt/voltana/infra/systemd/voltana.service /etc/systemd/system/
systemctl daemon-reload
systemctl enable voltana
```

### B8. First Deploy

```bash
cd /opt/voltana
bash scripts/deploy.sh
```

The script runs: `git pull` → frontend build → nginx config gen → DB migrations (000001–000013) →
`docker compose up -d --build api nginx` → nginx reload.

### B9. Post-Deploy Verification

```bash
# Health
curl https://your_domain.com/health
# → {"status":"ok"}

# HTTP redirect
curl -I http://your_domain.com/
# → HTTP/1.1 301

# Security headers present
curl -sI https://your_domain.com/health | grep -E "Strict-Transport|X-Frame|X-Content"

# OTP config endpoint (public)
curl https://your_domain.com/auth/otp/config
# → {"delivery_method":"deeplink"}

# SPA loads
curl -s https://your_domain.com/ | grep -i "voltana\|<title>"
```

### B10. Bootstrap Admin

```bash
docker exec voltana-postgres sh -c \
  'psql -U "$POSTGRES_USER" "$POSTGRES_DB" \
   -c "UPDATE users SET is_admin=true WHERE email='"'"'your@email.com'"'"';"'
```

### B11. Start Stack as Systemd Service

```bash
systemctl start voltana
systemctl status voltana
# → active (exited)
```

### B12. Backup Timer

```bash
cp /opt/voltana/infra/systemd/voltana-backup.service /etc/systemd/system/
cp /opt/voltana/infra/systemd/voltana-backup.timer   /etc/systemd/system/
systemctl daemon-reload
systemctl enable --now voltana-backup.timer

# Confirm timer
systemctl list-timers voltana-backup.timer
```

Run a manual backup to verify:

```bash
sudo -u voltana bash /opt/voltana/scripts/backup-db.sh
ls -lh /var/lib/voltana/backups/
```

### B13. VPS Feature Verification

Repeat the A3 curl tests against `https://your_domain.com` with real HTTPS.
