## Architecture Notes (from architect)

Go project structure to create:
```
voltana-api/
  cmd/server/main.go
  internal/
    domain/
    repository/
    service/
    handler/
      health_handler.go
    middleware/
  migrations/
  Dockerfile
  go.mod
  go.sum
```

Docker Compose service order:
1. postgres (health check: pg_isready)
2. redis (health check: redis-cli ping)
3. migrate (depends_on: postgres healthy)
4. api (depends_on: migrate complete, redis healthy)
5. nginx (depends_on: api healthy)
